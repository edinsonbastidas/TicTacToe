# Proyecto TicTacToe - Jakarta EE

Este proyecto es un juego de Tres en Raya desarrollado como un microproyecto web utilizando Java Servlets, JSP y JSTL, desplegado en un servidor WildFly.

## Características
* Tablero dinámico de 3x3.
* Juego contra el computador.
* Interfaz basada en imágenes (X, O y celdas vacías).
